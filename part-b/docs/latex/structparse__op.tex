\doxysubsection{Detailed Description}
Represents a parsed command operation.

This structure holds the details of a command parsed from an input string. It encapsulates the command, key, and value parts of the operation, as well as a generated response string in RESP format (used in communication protocols).

Members\+:
\begin{DoxyItemize}
\item cmd\+: The command name (e.\+g., "{}\+SET"{}, "{}\+GET"{}, "{}\+DEL"{}, etc.) indicating the operation.
\item key\+: The target key on which the command operates.
\item value\+: The associated value for operations that require it (e.\+g., "{}\+SET"{}); empty otherwise.
\item resp\+\_\+str\+: The response string generated following the parsing of the command, formatted as a RESP message. 
\end{DoxyItemize}
