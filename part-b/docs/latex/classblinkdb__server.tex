\doxysubsection{Detailed Description}
Redis-\/compatible server implementation

This class implements a server that can handle multiple client connections concurrently using {\bfseries{epoll}} for {\bfseries{non-\/blocking I/O}}. It interprets {\bfseries{RESP2}} protocol commands and interfaces with a blinkdb database instance for data storage and retrieval.

Supported commands\+:
\begin{DoxyItemize}
\item SET\+: Store a key-\/value pair
\item GET\+: Retrieve a value by key
\item DEL\+: Remove a key-\/value pair
\item QUIT\+: Disconnect client
\item CONFIG\+: Server configuration (basic implementation)
\item PRINT\+: Debug command to display database contents 
\end{DoxyItemize}
