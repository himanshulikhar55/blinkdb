Detailed Description

Defines the blinkdb_client class for connecting to a BlinkDB server and sending requests.

This file provides a simple client interface that provides public methods to either directly execute commands on the passed on BLINKDB instance based on the inputs or uses POSIX sockets to establish a TCP connection to a BlinkDB server, send requests, and receive responses. The former is used for local mode and the latter for client-server mode.
