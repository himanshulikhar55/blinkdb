Detailed Description

This is the diskstorage class. It is used to store key-value pairs on disk. It writes the key-value pairs to a file in append mode and reads them back using a sparse index.

diskstorage::diskstorage ( )

Construct a new diskstorage object. It opens the file "data.log" in current directory in append mode. It currently stores data as key-value pairs in the file (space-separated).
