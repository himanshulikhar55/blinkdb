\doxysubsection{Detailed Description}
Implementation of a Redis-\/like server using epoll for non-\/blocking I/O. 

This file defines the \doxylink{classblinkdb__server}{blinkdb\+\_\+server} class which implements a server that\+:
\begin{DoxyItemize}
\item Uses epoll for efficient handling of multiple client connections
\item Implements basic Redis-\/compatible commands (SET, GET, DEL, QUIT)
\item Parses RESP (Redis Serialization Protocol) commands
\item Provides connection management and data storage via blinkdb
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Default server port is 9001 
\end{DoxyNote}
